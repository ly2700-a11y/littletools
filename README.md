## 学习桌宠监督器（桌面小组件）

一个桌面小组件风格的小工具，包含：

- 桌宠状态反馈（平静、警告、高压监督）
- 番茄计时（25 分钟专注 / 5 分钟短休 / 15 分钟长休）
- 摸鱼检测（学习计时中离开标签页超过 12 秒会计入警告）
- 本地数据保存（完成轮次、警告次数）
- 小窗控制（置顶、最小化、关闭）

### 使用方式

1. 安装依赖：`npm install`
2. 启动小组件：`npm start`
3. 选择计时模式并点击“开始”
4. 学习时尽量不要让窗口失焦过久，桌宠会监督你

### 文件说明

- `index.html`：页面结构
- `style.css`：桌宠与界面样式
- `app.js`：计时逻辑、监督逻辑、数据存储
- `main.js`：Electron 主进程（窗口创建与控制）
- `preload.js`：渲染进程与主进程通信桥
